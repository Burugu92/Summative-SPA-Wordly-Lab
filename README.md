# Summative-SPA-Wordly-Lab

Wordly is a Single Page Application (SPA) dictionary built with HTML, CSS, and JavaScript. It allows users to search for words, view definitions, synonyms, pronunciation, and listen to audioâ€”all without page reloads. This project demonstrates modern web development practices, including API integration, dynamic DOM manipulation, and responsive design.

#Features

Word Search: Users can enter a word to fetch definitions from a public dictionary API.
Dynamic Data Display: Displays pronunciation, part of speech, definitions, synonyms, and audio playback.
Error Handling: Gracefully handles invalid words, missing data, and API errors.
Responsive Design: Works on both desktop and mobile devices.
Interactive UX: Seamless updates without refreshing the page.

#Technologies Used

HTML: Semantic markup for structure and accessibility.
CSS: Styling and layout for an appealing, user-friendly interface.
JavaScript:
Fetch API for retrieving word data from the Free Dictionary API.
DOM manipulation for dynamic content updates.
Event listeners for form submissions and user interactions.
